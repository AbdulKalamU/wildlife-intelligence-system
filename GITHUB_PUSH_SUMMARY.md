## 🔍 What's NOT in Git (By Design)

These files are ignored and won't be pushed:
- ❌ `*.db` - Database files (runtime data)
- ❌ `*.pt` - Model files (downloaded automatically)
- ❌ `__pycache__/` - Python cache
- ❌ `venv/` - Virtual environment
- ❌ `.DS_Store` - macOS system files

**Why?** These are generated at runtime or environment-specific.
